Bank Marketing Campaign Analysis
Project Overview
This project involves analyzing a bank's direct marketing campaign data to predict whether a client will subscribe to a term deposit (variable y). The dataset contains information about client demographics, previous contacts, and social-economic context.

Dataset Description
The data is sourced from the UCI Machine Learning Repository. It contains two main files:

bank-full.csv: Contains all 45,211 examples and 17 inputs, ordered by date.

bank.csv: A smaller sample (10% of the data) for testing and faster computation.

Key Features:
Demographics: Age, Job, Marital Status, Education.

Financials: Default status, Average yearly balance, Housing loan, Personal loan.

Campaign Info: Contact type, Day/Month of last contact, Duration of contact.

Previous Contacts: Number of contacts performed during this campaign and previous campaigns.

Target (y): Has the client subscribed a term deposit? (binary: 'yes', 'no').
